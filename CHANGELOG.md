# Changelog

## v1.0.1 — Pilot-name hotfix

- Prevent gameplay and menu shortcuts from intercepting keystrokes while a
  text field is focused.
- Pilot names can now include every permitted letter, including control keys
  such as `P`, `C`, `F`, `X`, `W`, `A`, `S`, and `D`.

## v1.0.0 — Launch

- Complete single-file neon arcade survival game.
- Multi-wave enemy progression and recurring Scrap Warden boss encounters.
- Asteroid, shard, pulse, dash, upgrade, and active-effect combat systems.
- Secure global leaderboard with verified run submission.
- Responsive top and bottom safe-zone HUDs, animated menus, and visual polish.
- Built-in release update notification and direct single-file downloads.
